# 📊 Formulário de Satisfação
Este é um projeto desenvolvido em **React** para a disciplina de Tecnologia de Desenvolvimento de Interface Grafica.

## 🚀 Tecnologias Utilizadas
* **React** (Vite)
* **React Router Dom** (Navegação entre páginas)
* **CSS3** (Estilização personalizada e Responsividade)
* **Formspree** (Integração para recebimento de e-mails)

## 🛠️ Funcionalidades
- [x] Navegação entre Home, Sobre e Formulário.
- [x] Validação básica de campos.
- [x] Design responsivo com foco em UX.
- [x] Página de Erro 404 personalizada.
